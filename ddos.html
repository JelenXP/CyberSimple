DDoS

Přetížení služby velkým množstvím požadavků.

Co to je

DDoS (Distributed Denial of Service) je útok, kdy útočníci zaplaví server obrovským množstvím požadavků, aby jej zpomalili nebo zcela vyřadili z provozu.

Jak funguje

Botnety generují tisíce až miliony požadavků na cílový server současně.

Jak poznat

Pomalejší web, časté výpadky, nemožnost se připojit k službě.

Jak se bránit

Používat ochranu proti DDoS od poskytovatele, firewall, load balancery a monitorování síťového provozu.
