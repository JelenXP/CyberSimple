Watering‑hole

Napadení oblíbených webů cílové skupiny.

Co to je

Útočník kompromituje web často navštěvovaný obětí, aby šířil malware nebo kradl data.

Jak funguje

Web vypadá normálně, ale obsahuje škodlivý skript, který napadne návštěvníka.

Jak poznat

Někdy se projeví výstrahami antiviru, pomalým načítáním nebo neočekávanými chybami.

Jak se bránit

Používat antivir, aktualizovat prohlížeč, blokovat podezřelé skripty a sledovat bezpečnostní oznámení webu.
